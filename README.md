# MFO Parser Service

**Парсер данных микрофинансовых организаций с сайта [vsezaimyonline.ru](https://vsezaimyonline.ru) без использования API**

Проект демонстрирует навыки **браузерного парсинга сложных сайтов**, работы с современным стеком и хранения структурированных данных.

### Что парсит проект

Основная страница компании (например `/zaymer`, `/ekapusta` и др.):

- Основной заголовок и lead-текст  
- Блок «О компании»  
- Преимущества и недостатки (из аккордеонов/спойлеров)  
- Инструкция «Как получить займ» (шаги из списка)  
- Все FAQ и аккордеоны на странице  

Результат сохраняется в PostgreSQL в виде нормализованного JSON-объекта.

### Стек технологий

- **Backend**: NestJS + TypeScript  
- **Парсинг**: Playwright (Chromium headless)  
- **База данных**: PostgreSQL + Prisma 7 (с `@prisma/adapter-pg`)  
- **Валидация**: Zod  
- **Контейнеризация**: Docker + docker-compose  

### Как запустить

#### Локально

```bash
# Установка зависимостей
npm install

# Генерация Prisma-клиента
npx prisma generate

# Создание/синхронизация базы данных
npx prisma db push
# или для миграций:
# npx prisma migrate dev --name init

# Запуск в режиме разработки
npm run start:dev
